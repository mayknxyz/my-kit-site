---
title: Tailwind CSS Skill
description: Domain skill for Tailwind CSS projects.
---

The Tailwind CSS skill gives Claude deep knowledge of utility-first CSS patterns and Tailwind's configuration system.

## When It's Active

Activated automatically when `/mykit.init` detects Tailwind CSS (presence of `tailwind.config.*` or `@tailwindcss` in dependencies).

## What It Teaches Claude

- Utility class composition and ordering
- Custom theme configuration
- Responsive and state variant patterns
- Plugin authoring
- CSS-in-JS escape hatches
