## `/mykit.plan`

The plan command reads the current spec and generates a detailed implementation plan. It identifies files to create or modify, dependencies, and the order of operations.

### Usage

```bash
/mykit.plan
```

### What It Produces

A plan document covering:
- Files to create, modify, or delete
- Implementation steps in order
- Key decisions and trade-offs
- Risks and fallback approaches
