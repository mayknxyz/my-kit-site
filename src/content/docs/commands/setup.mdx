## `/mykit.init`

Initialize my-kit in the current project. Sets up project-level configuration and detects your stack.

```bash
/mykit.init
```

## `/mykit.sync`

Sync my-kit's configuration with the latest version. Useful after upgrading my-kit or changing your project's stack.

```bash
/mykit.sync
```
