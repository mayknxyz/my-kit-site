## `/mykit.specify`

Create a structured specification for a feature, bug fix, or task.

```bash
/mykit.specify
```

See [Workflow: Specify](/workflow/specify/) for details.

## `/mykit.plan`

Generate an implementation plan from the current specification.

```bash
/mykit.plan
```

See [Workflow: Plan](/workflow/plan/) for details.

## `/mykit.tasks`

Break the implementation plan into trackable tasks.

```bash
/mykit.tasks
```

See [Workflow: Tasks](/workflow/tasks/) for details.

## `/mykit.implement`

Execute tasks from the task list, writing code and running tests.

```bash
/mykit.implement
```

See [Workflow: Implement](/workflow/implement/) for details.
