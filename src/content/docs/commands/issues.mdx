## `/mykit.issue.create`

Create a new GitHub issue with structured content.

```bash
/mykit.issue.create
```

## `/mykit.issue.edit`

Edit an existing GitHub issue.

```bash
/mykit.issue.edit
```

## `/mykit.issue.view`

View details of a GitHub issue.

```bash
/mykit.issue.view
```

## `/mykit.issue.list`

List GitHub issues with filtering options.

```bash
/mykit.issue.list
```

## `/mykit.label.sync`

Sync GitHub labels to match my-kit's standard label set.

```bash
/mykit.label.sync
```
